# AION Guard Pro — Commercial Preview (v2)

## Purpose

This document describes the commercial direction of AION Guard Pro.

It is intended for:

- founders
- integrators
- treasury operators
- strategic partners
- prospective investors

This file is a positioning preview, not a release of private implementation.

---

## Core framing

AION Guard Pro is the commercial operating layer built on top of the Wallet Lite execution-governance foundation.

Wallet Lite establishes the primitive:

- governed execution backend
- local or deployable runtime
- API called before execution
- evidence and receipt engine
- adapter-ready integration surface

AION Guard Pro extends that primitive into a stronger operational, intelligence, and governance system for serious operators.

---

## Lite vs Pro

### Wallet Lite

Wallet Lite is the foundational execution-governance primitive.

It provides:

- governed execution posture
- deterministic decision outputs
- reason-coded review and blocking behavior
- receipt and evidence generation
- adapter-facing backend architecture
- packaged runtime deployment path
- proof-backed execution semantics

Wallet Lite is designed to be the authoritative kernel that others can build on top of.

### Guard Pro

Guard Pro is the premium operating layer.

It expands Wallet Lite into a stronger deployment surface for:

- higher-assurance treasury operations
- richer policy and guardian workflows
- stronger intelligence and simulation
- premium review and operational tooling
- commercial deployment and integration needs

## Strategic logic

The public strategy is:

1. demonstrate the primitive
2. demonstrate the proof surface
3. demonstrate the governance model
4. expose a clean integrator-facing architecture
5. expand commercially through deeper control layers and premium operator tooling

This allows AION to preserve a strong closed-core moat while still letting the market see that the primitive is real.
