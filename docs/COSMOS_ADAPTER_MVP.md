# Cosmos Adapter MVP

## Purpose
This module adds the first Cosmos-native protection surface to AION Wallet-Lite.

## MVP goals
- accept Cosmos preview input
- classify message/module patterns
- infer narrative
- compute basic risk
- map to AION decision shape
- support fixture selftests

## Initial supported cases
- bank send
- staking / delegation
- governance vote / governance-sensitive action
- IBC transfer
- unknown wasm / module review

## Public positioning
This is the first Cosmos-native adapter scaffold for AION.
It extends Wallet-Lite beyond EVM, Bitcoin MVP, and Solana MVP toward broader digital asset protection.
