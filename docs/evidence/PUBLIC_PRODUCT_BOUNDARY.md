# Public Product Boundary

## Public-safe contents

This public repository is intended to expose:

- proof and evidence artifacts
- validation declarations
- public capability boundaries
- investor-safe and partner-safe product claims

## Excluded from public repo

The public repository must not expose:

- runtime implementation internals
- build or execution scripts
- private harnesses
- internal transcripts
- source or built engine logic not intended for public release
- local environment details not required for proof

## Boundary statement

This repo is evidence-first, not implementation-first.

Its purpose is to prove that Wallet Guard Lite works, not to disclose the private implementation required to reproduce or clone it.
