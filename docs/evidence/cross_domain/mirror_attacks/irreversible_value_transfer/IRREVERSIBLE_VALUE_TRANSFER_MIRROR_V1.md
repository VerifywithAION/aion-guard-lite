# Irreversible Value Transfer Mirror V1

## Purpose

Show that multiple domains can express the same deeper invariant:

**irreversible external value transfer or irreversible loss event**

The surface forms differ.
The consequence family does not.

---

## Mirror cases

### AI agents
#### Surface action
Agent sends 5 ETH to an external wallet.

#### Deep pattern
Irreversible external value transfer.

#### Expected governance
- decision: `BLOCK`
- gate: `STOP`

---

### Enterprise approvals
#### Surface action
Single signer approves a large external vendor payout.

#### Deep pattern
Irreversible external value transfer.

#### Expected governance
- decision: `REVIEW`
- gate: `QUEUE`

---

### Machine-to-machine execution
#### Surface action
Service transfers value to an unknown endpoint.

#### Deep pattern
Irreversible external value transfer.

#### Expected governance
- decision: `BLOCK`
- gate: `STOP`

---

### Trading systems
#### Surface action
Oversize order creates liquidation exposure.

#### Deep pattern
Irreversible loss event.

#### Expected governance
- decision: `BLOCK`
- gate: `STOP`

---

## Significance

These systems look unrelated on the surface.

But all of them express a closely related invariant:

**an action may create irreversible value loss or transfer beyond safe policy boundaries**

That is why governed execution should reason at the consequence layer, not only the domain layer.
