# Wallet Lite - Screenshot Evidence (C5)

## Purpose

This folder contains curated visual artifacts that support the Wallet Lite proof narrative.

These images are not the canonical source of truth.

They are illustrative complements to the real proof system.

---

## Important distinction

Canonical proof lives in:

- docs/evidence/validation/
- reproducible scripts
- JSON proof artifacts
- deterministic verification flows

Screenshots are:

- non-authoritative
- non-replayable
- human-readable context only

---

## What these screenshots show

When present, screenshots may illustrate:

- Sepolia policy verification success
- governed preview responses in harness UI
- C2 proof pack terminal output
- wallet interaction flows (MetaMask / Rabby)

---

## What they do NOT represent

Screenshots do not replace:

- reproducible verification
- machine-readable evidence
- receipt artifacts
- policy enforcement logic

---

## Recommended usage

Use screenshots to:

- quickly understand system behavior
- visually confirm expected flows
- support presentations or public explanations

Use proof artifacts to:

- verify correctness
- audit behavior
- reproduce results independently
