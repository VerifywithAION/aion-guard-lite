# Wallet Guard Lite — Founder Launch Note V1

Wallet Guard Lite exists for one reason:

to reduce the chance that crypto users lose money because they approve, sign, or send something unsafe.

The product is intentionally simple at the surface:

- call an API
- evaluate a transaction before execution
- return ALLOW, WARN, or BLOCK

Under that simple surface, the goal is much bigger:

to make transaction safety easy enough that wallets and platforms can add it without rebuilding their whole product.

This public repo is evidence-first.

It shows proof, validation, attack-lab results, and defensive hardening results.

It does not expose the private implementation depth required to clone the core system.

Wallet Guard Lite is the public protection layer.

Wallet Guard Pro is the future private intelligence layer for serious platforms.
